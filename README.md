# Space Defence

A small space shooter made to learn monogame.

## Controls
`esc` opens the pause menu. Pressing it again will always close the game.

Movement:
WASD are used. Use W to engage thrusters and accelerate forwards. A and D turn left and right respectively.
To break, turn around, and use W to slow down. This can be made more easily by using S, which rotates the ship backwards.

Shooting:
Use the mouse to aim and click to shoot. With the double gun, it is possible to hold the mouse button down to shoot rapidly without getting a repetitive strain injury.

## Notes to teacher:
- Weapons are upgraded by picking up crates
- The button hints (`(A) Play`) indicate the button to press on controller, use the mouse to click the text instead

## Known bugs:
- Going back to the main menu makes visual buttons freeze. For now the workaround to this is closing the game with Escape.
